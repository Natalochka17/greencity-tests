# Test Cases for GreenCity Events Page

## Test Case 1: Перевірка відкриття сторінки новин

### Title
Перевірка успішного відкриття сторінки новин GreenCity

### Preconditions
- Користувач має доступ до інтернету
- Відкритий браузер

### Test Steps

| Step | Action | Data | Expected Result |
|------|--------|------|----------------|
| 1 | Відкрити браузер | - | Браузер відкрився |
| 2 | Ввести адресу сторінки у адресний рядок | https://www.greencity.cx.ua/#/greenCity/events | Сторінка починає завантажуватися |
| 3 | Дочекатися повного завантаження сторінки | - | Відображається сторінка новин GreenCity |
| 4 | Перевірити наявність списку новин | - | На сторінці відображаються картки новин |

## Test Case 2: Перевірка відкриття детальної сторінки новини

### Title
Перевірка переходу до детальної інформації про новину

### Preconditions
- Користувач знаходиться на сторінці новин GreenCity
- На сторінці відображається хоча б одна новина

### Test Steps

| Step | Action | Data | Expected Result |
|------|--------|------|----------------|
| 1 | Вибрати будь-яку новину зі списку | Перша новина | Новина підсвічується або стає доступною для відкриття |
| 2 | Натиснути на назву або зображення новини | - | Відкривається сторінка вибраної новини |
| 3 | Перевірити наявність заголовка новини | - | На сторінці відображається заголовок новини |
| 4 | Перевірити наявність тексту новини | - | Відображається повний текст або опис новини |

### Title

### Preconditions
- Відкрита детальна сторінка будь-якої новини

### Test Steps

| Step | Action | Data | Expected Result |
|------|--------|------|----------------|
| 1 | Натиснути кнопку браузера "Назад" | - | Користувач повертається на сторінку новин |
| 2 | Перевірити список новин | - | Відображається той самий список новин, що був до відкриття новини |
